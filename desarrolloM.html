Desarrollo

El desarrollo de MenuetOS se ha centrado en una implementación rápida, simple y eficiente. MenuetOS tiene capacidades de red y una pila TCP / IP que funciona . La mayor parte del código de red está escrito por Mike Hibbett. El enfoque principal de Menuet ha sido crear un entorno para una fácil programación de ensamblajes, pero aún es posible ejecutar software escrito en lenguajes de programación de alto nivel en el núcleo del ensamblador. El mayor esfuerzo individual hacia el soporte de lenguaje de alto nivel es el trabajo de Jarek Pelczar en la conversión de bibliotecas C a Menuet. La GUI en la versión 0.99 admite resoluciones de pantalla de hasta 1920 x 1080 (16 millones de colores) con transparencia de ventana. El sistema operativo tiene soporte para varias clases de periféricos USB 2.0 . MenuetOS viene con las versiones shareware de Quake and Doom . Para el almacenamiento en disco, MenuetOS es compatible con el sistema de archivos FAT32 .
